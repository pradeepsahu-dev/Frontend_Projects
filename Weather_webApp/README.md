# 🌤 Weather App

A simple weather app that shows real-time weather using API.


## 🚀 Features

* Search weather by city name
* Shows temperature, humidity, and wind speed
* Dynamic weather icons
* Error message for wrong location

## 🛠 Tech Used

* HTML
* CSS
* JavaScript
* OpenWeather API


## ▶️ How to Use

1. Enter city name
2. Click search button
3. View weather details

# 📝 Task Manager (Todo App)

A simple and clean todo app to manage daily tasks easily.

# 🚀 Features

* Add new tasks
* Delete tasks
* New tasks appear at the top
* Add task using Enter key

# 🛠 Tech Used

* HTML
* CSS
* JavaScript

# ▶️ How to Use

1. Write your task
2. Click "Add Task" or press Enter
3. Manage your tasks easily

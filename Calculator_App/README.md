# 🧮 Calculator App (CalcPro)

A simple and modern calculator built using "HTML, CSS, and JavaScript".
with a clean UI and keyboard support.

# 🚀 Features

* ➕ Basic operations (Addition, Subtraction, Multiplication, Division)
* 🧹 Clear (C) button to reset
* ⌫ Delete (DEL) button
* 🧠 Real-time calculation using JavaScript
* ⌨️ Keyboard support (Enter, Backspace, Numbers, Operators)
* 🎨 Modern UI with glass/soft design

# 🛠 Tech Used

* HTML
* CSS
* JavaScript

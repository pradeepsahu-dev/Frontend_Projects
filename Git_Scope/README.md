# 🔍 GitScope

A modern GitHub Profile Finder that fetches real-time user data using GitHub API.

# 🚀 Features

* Search GitHub users instantly
* Shows profile image, bio, followers & following
* Displays public repositories
* Scrollable repositories list
* Enter key support
* Dynamic UI with API data

# 🛠 Tech Used

* HTML
* CSS
* JavaScript
* GitHub REST API

# ▶️ How to Use

1. Enter GitHub username
2. Click search button
3. View profile details and repositories

# ✊ Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game built using "HTML, CSS, and JavaScript".
This project uses random computer choices and real-time result updates with a clean UI design.

# 🚀 Features

* ✊ Rock, 📃 Paper, ✂️ Scissor gameplay
* 🤖 Random computer move generation
* 🎯 Instant win/lose/draw result
* 🖥 Dynamic DOM updates using JavaScript
* 🎨 Simple and responsive UI
* ⚡ Interactive button effects

# 🛠 Tech Used

* HTML
* CSS
* JavaScript

# 🎮 Game Rules

* Rock beats Scissor
* Paper beats Rock
* Scissor beats Paper
